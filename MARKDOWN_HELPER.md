# Справочник по разметке и коду для MoonAI

В этом файле показано, как красиво оформлять сообщения и код в чате MoonAI.

## Что поддерживается

- Заголовки: `#`, `##`, `###`
- Полужирный текст: `**жирный текст**`
- Встроенный код: `` `код` ``
- Блоки кода: [code][/code]
- Таблицы: строки с `|`
- Маркированные списки: `- пункт`

## Примеры

### Заголовок

# Заголовок первого уровня
## Заголовок второго уровня
### Заголовок третьего уровня

### Полужирный текст

Это **важный текст**, который выделяется.

### Встроенный код

Используйте `console.log('Hello')` для вывода в консоль.

### Блок кода

```ts
function greet() {
  console.log('Hello, MoonAI!');
}
```

### Список

- Первый пункт
- Второй пункт
- Третий пункт

### Таблица

| Элемент | Описание |
|---|---|
| `code` | Встроенный код |
| ` ``` ` | Блок кода |

### Рекомендация

Если вы хотите выделить код, используйте три обратные кавычки и укажите язык:

```python
print('Hello, MoonAI!')
```
